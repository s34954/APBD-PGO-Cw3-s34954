# System kawiarni – ćwiczenie 3

Prosty program konsolowy napisany w Javie. Symuluje składanie zamówienia w małej kawiarni.

## Co robi program

* tworzy kilka produktów (kawa, tiramisu, ciasteczko)
* tworzy klienta (Tony Stark)
* dodaje produkty do zamówienia
* liczy łączną wartość zamówienia
* pokazuje liczbę produktów
* pozwala oznaczyć zamówienie jako opłacone
* porównuje produkty za pomocą equals()

## Struktura

* ProduktMenu – dane o produkcie
* KlientKawiarni – dane klienta
* Zamowienie – obsługa zamówienia
* Main – uruchomienie programu

## Jak uruchomić

Uruchomić klasę Main w IntelliJ lub innym IDE.

## Uwagi

Projekt zrobiony na potrzeby ćwiczeń, skupia się na podstawach:
klasy, ArrayList, metody statyczne, toString() i equals().
